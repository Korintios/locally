# Locally

Locally se centra en la **interacción del usuario con la inteligencia artificial (IA)** y una interfaz de chat funcional.

* **Funcionalidad de Chat:** La aplicación permite a los usuarios **enviar mensajes de texto, imágenes y archivos** a la inteligencia artificial para interactuar con ella.
* **Gestión de MVPs:** Incluye un botón dentro de la interfaz de chat para **activar o desactivar otros MVPs**, ofreciendo flexibilidad en las funcionalidades.

### Barra Lateral (Sidebar) y Navegación

La **barra lateral** está diseñada para una navegación eficiente y una gestión clara del historial de conversaciones, además de ser **colapsable** para optimizar el espacio de la pantalla.

El contenido de la barra se estructura en tres secciones principales:

* **Encabezado (Header):**
    * Botón para **iniciar un nuevo chat**.
    * Botón de **búsqueda** para filtrar el historial de conversaciones.
* **Cuerpo (Body):**
    * **Historial de chats** guardados.
* **Pie de página (Footer):**
    * Botón de **enlace a GitHub** para soporte o referencia.

### Barra de Chat y Entrada de Mensajes
La **barra de chat** en la parte inferior de la interfaz permite a los usuarios **escribir y enviar mensajes** fácilmente. Incluye:
* **Campo de entrada de texto** para redactar mensajes.
* **Botón para adjuntar imágenes y archivos**.
* **Botón de envío** para enviar el mensaje a la IA.
* **Botón de configuración** al modal de configuración y MCPs.
* Indicadores visuales para mostrar el estado de envío y recepción de mensajes.
* Chat en tiempo real con actualizaciones instantáneas.

### Popovers

#### Popover de Gestión de MVPs
El **popover de gestión de MVPs** (se abre al presionar el botón de MVPs en la barra de chat) permite a los usuarios activar o desactivar funcionalidades adicionales del MVP según sus necesidades y abrir el modal de configuración para ajustes más detallados.

#### Popover de opciones de chat
El **popover de opciones de chat** (se abre al presionar el botón de opciones en los distintos historiales de chat) ofrece a los usuarios acceso rápido a configuraciones específicas del chat, como borrar el chat y renombrarlo.

### Modales

#### Modal de Configuración
El **modal de configuración** (se abre al presionar el botón de configuración en el Popover de la barra de chat) permite a los usuarios ajustar las preferencias de la aplicación, como la selección del modelo de IA y configuración de MCPs.
